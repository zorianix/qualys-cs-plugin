A comma-separated list of Software names that to be evaluated for build failure. It can be a simple comma-separated list with or without specific version. Software names with version should be provided in format - Softwarename = eg. rpm-libs=4.8.0-55.el6.
To address multiple packages with a similar name using input in regex format is also accepted in this field. Click here to get a valid regex string. Example - .*=1.*, =0.*, .*
